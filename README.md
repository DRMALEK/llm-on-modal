# llm-on-modal
Tutorial series: self-host open-source LLMs on Modal.com with vLLM serving

Part 1 – Deploy Your First LLM on Modal: Basic Modal app that loads an open-source LLM (e.g. a quantized Mistral or Llama model) from HuggingFace and exposes it as a private GPU-backed endpoint.

Part 2 – High-Throughput Serving with vLLM on Modal: Upgrade Part 1 to use vLLM's AsyncLLMEngine inside a Modal app, with streaming support and concurrent request handling.

Part 3 – Deploy a Private Vision-Language Model (VLM): Extend the vLLM Modal setup to serve a VLM (e.g. LLaVA or Qwen-VL), handling image inputs alongside text prompts.
